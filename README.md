## Requires

- [Neovim](https://neovim.io/) (>= 0.8)
- GCC or Clang (for treesitter)
- Python
- Npm

## Install

```text
git clone https://github.com/SHTRAMPANTUNC/MyNvim
```

## Move Files

```text
mv MyNvim/* ~/.config/nvim
```
